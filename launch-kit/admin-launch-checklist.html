Admin Launch Checklist

Use this before inviting a full team so parent access, player rosters, and game-day tracking are clean from the start.

1. Create Team

  1. Sign in with the approved admin account.
  2. Open Manage, then Manage Teams.
  3. Create or select the team.
  4. Copy the team code for parent requests.

2. Preload Roster

  1. Add each player by full name and jersey number.
  2. Select one or more positions if needed.
  3. Use Sync after roster edits.
  4. Review unclaimed players before game day.

3. Invite Parents

  1. Send the parent email or team chat post.
  2. Include the correct team code.
  3. Tell parents to enter their child's jersey number.
  4. Remind iPhone users to Add to Home Screen.

4. Approve Requests

  1. Open Manage Teams as admin.
  2. Review pending parent requests.
  3. Approve only matching team and jersey requests.
  4. Ask parents to sign out/in or Sync if needed.

Best practice: preload the roster first, then share the team code. Parents should not create teams. They request access, verify a child by jersey number, and then see only approved player/team contexts.
