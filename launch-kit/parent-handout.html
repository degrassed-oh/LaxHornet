Fast lacrosse stats for parents, coaches, and families.

Track multiple players, save games, review season totals, and share live game updates from another phone, tablet, or computer.

Scan to open
https://degrasse-mastermind.github.io/LaxHornet/

Install On iPhone

  1. Open the link in Safari.
  2. Tap the Share button.
  3. Tap Add to Home Screen.
  4. Open LaxHornet from the new icon.

Start Tracking

  1. Create a User Profile or sign in.
  2. Add one or more players in Player Settings.
  3. Tap the player you want active.
  4. Start a game and use the big stat buttons.

What It Tracks

  • Goals, assists, shots, saves, faceoffs, ground balls, clears, turnovers, penalties, and notes.
  • Game Impact Score and per-player season dashboard.
  • Post-game event edits, tags, and corrections.

Share Live

  • During a game, tap Live Share beside the game date.
  • The link copies automatically.
  • Paste it into a text or email so family can watch live.

Built For Game Day

LaxHornet is mobile-first, works offline, and keeps each user's players and games separate when signed in. It is designed for quick one-handed tracking from the sideline.
